发表于 2018-12-01 | 分类于 安装说明

1. 需要的环境

  • npm
  • github账号
  • git

2. 安装hexo步骤

  • hexo的安装大致参考这里，其中几个重要的问题和解决方式如下
    • npm安装速度慢

      执行 npm config set registry https://registry.npm.taobao.org
      这里采用淘宝的镜像，设置之后安装速度快多了

    • 执行的命令

      创建blog文件夹，并进入文件夹
      输入npm install hexo -g
      输入hexo init
      输入npm install，安装所需要的组件
      输入hexo g，首次体验Hexo
      输入hexo s，开启服务器，访问该网址，正式体验Hexo

    • git中公钥问题
      我的公钥之前用码云的时候生成过一次，然后直接找到 GitHub账号下的setting找到SSH keys and GPG keys选项，设置SSH keys即可，然后执行

      ssh -T git@github.com ps:在git bash命令行下 执行该命令

    • 修改配置 (blog/_config.yml)

3. 安装next主题步骤

  • 在blog/themes目录下 执行

    git clone https://github.com/iissnan/hexo-theme-next themes/next
    项目下载到themes后确保next和landscape文件夹是同级的

  • 修改blog/_config.yml中配置

  • 修改E:/blog/themes/next

  • tags 创建步骤

    1. 创建一个新的页面，命名为tags，执行

      hexo new page ‘tags’

    2. 编辑tags 文件夹下的index.md

  • categories 创建步骤

    1. 创建一个新的页面，命名为categories，执行

      hexo new page ‘categories’

    2. 编辑categories 文件夹下的index.md
